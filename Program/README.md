## ğŸ”§ Troubleshooting

### Common Issues

```

```

```
Solution: Increase MAX_VIDEO_DURATION in .env
Or process shorter segments
```

## ğŸ¨ Design Decisions

### Why Whisper?
- Open-source and free
- State-of-the-art accuracy
- Multilingual support
- Local processing (no API costs)

### Why Google Gemini?
- Free tier with generous limits
- Excellent at structured output
- Fast response times
- Good educational content understanding

### Why FastAPI?
- Modern async support
- Automatic API documentation
- Easy integration with Python ML stack
- Excellent performance

### Segmentation Strategy
- **Short videos (<2000 words)**: Process as whole for better context
- **Long videos**: 5-minute time-based segments to respect token limits

## ğŸ“Š Performance

- **Whisper Base Model**: ~1x realtime (10 min video = 10 min processing)
- **Gemini API**: ~5-10 seconds per segment
- **Total**: ~1.5-2x video duration for complete pipeline

**Optimization tips:**
- Use `tiny` or `base` Whisper models for speed
- Enable GPU for 3-5x faster transcription
- Process shorter videos for quicker results

## ğŸš§ Limitations

- English videos work best (Whisper supports others)
- Very long videos (>2 hours) may take significant time
- Requires stable internet for Gemini API
- Quality depends on audio clarity
